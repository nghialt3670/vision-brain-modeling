## Codebases & Libraries

| Repository | Description | Link |
|------------|-------------|------|
| **MindEye** | SOTA fMRI-to-image reconstruction and retrieval pipeline. | [GitHub](https://github.com/MedPhD-Lab/MindEye) |
| **MindEye2** | Multi-subject, contrastive fMRI visual decoding models. | [GitHub](https://github.com/MedPhD-Lab/MindEye2) |
| **ATM** | SOTA EEG-to-image zero-shot reconstruction. | [GitHub](https://github.com/ZhouYujia-tju/ATM) |
| **Brain Diffuser** | Latent diffusion visual reconstruction model. | [GitHub](https://github.com/ozcelikyilmaz/brain-diffuser) |
| **MNE-Python** | The primary Python library for processing EEG/MEG signals. | [mne.tools](https://mne.tools/) |
| **Nilearn** | Statistical learning on neuroimaging data. | [nilearn.github.io](https://nilearn.github.io/) |
