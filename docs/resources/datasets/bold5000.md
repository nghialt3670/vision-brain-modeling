# BOLD5000

> A large-scale slow-TR fMRI dataset bridging traditional neuroscience image sets with modern vision benchmarks.

---

## Overview

| Property | Value |
| :--- | :--- |
| **Modality** | fMRI (3 Tesla) |
| **Subjects** | 4 healthy adults |
| **Stimuli** | 5,254 images from COCO, ImageNet, and SUN databases |
| **TR** | 2 seconds (standard BOLD) |
| **Access** | Public — [bold5000.org](https://bold5000-dataset.github.io/website/) |
| **Paper** | Chang et al., *Scientific Data* 2019 — [DOI](https://doi.org/10.1038/s41597-019-0052-3) |

---

## Stimulus Composition

| Source | Images | Description |
| :--- | :--- | :--- |
| MS-COCO | 2,000 | Complex natural scenes with multiple objects |
| ImageNet | 1,000 | Single-object images across diverse categories |
| SUN Database | 1,000 | Scene images (indoor + outdoor environments) |
| Shared core | 1,254 | Seen by all subjects for cross-subject analysis |

---

## Design

Images were presented at 1 Hz (1 sec on, 1 sec off) while subjects maintained fixation. The multi-database composition makes BOLD5000 unique — it can simultaneously benchmark models trained on object, scene, and complex image distributions.

---

## Significance

- **Bridges ImageNet and scene-level representations** within a single dataset.
- Enables evaluation of models trained on different vision datasets without domain shift concerns.
- Used in generalization studies to test decoders beyond their primary training distribution.
