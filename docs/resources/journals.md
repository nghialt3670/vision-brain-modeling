# Journals

> Peer-reviewed journals and preprint servers for Vision-Brain Modeling and Computational Neuroscience.

## Peer-Reviewed Journals

| Journal | Publisher | Scope |
| :--- | :--- | :--- |
| **[Nature Neuroscience](https://www.nature.com/neuro/)** | Nature Portfolio | Foundational neuroscience discoveries |
| **[NeuroImage](https://www.sciencedirect.com/journal/neuroimage)** | Elsevier | Neuroimaging methods & brain decoding |
| **[Current Biology](https://www.cell.com/current-biology/home)** | Cell Press | Perception, cognition, visual neuroscience |
| **[Journal of Neuroscience](https://www.jneurosci.org/)** | Society for Neuroscience | Computational and systems neuroscience |
| **[Cerebral Cortex](https://academic.oup.com/cercor)** | Oxford Academic | Cortical representations and visual areas |
| **[PLOS Computational Biology](https://journals.plos.org/ploscompbiol/)** | PLOS | Open-access computational neuroscience |
| **[IEEE TNNLS](https://cis.ieee.org/publications/t-neural-networks-and-learning-systems)** | IEEE | Neural network methods incl. BCI decoding |
